st3-user
==========

Personal Sublime Text 3 User config directory

install
==========

[Download Sublime Text 3](http://www.sublimetext.com/3).

User directory
```sh
cd ~/.config/sublime-text-3/Packages/User
git init
git remote add -f origin git@github.com:IcanDivideBy0/st3-user.git
git checkout master
cd -
```
